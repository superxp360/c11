Week 1 - Day 1: Keyboard Shortcuts

Operating System

  • Command + C: Copy
  • Command + V: Paste
  • Command + Q: Quit
  • Command + W: Close Window
  • Command + Z: Undo AKA LIFE SAVER!!!!!!
  • Command + S: Save

Visual Studio Code(VSC)

  • Command + B: Toggle Sidebar
  • Command + J: Toggle Terminal AKA: Jarvis
  • Option + Z: Soft Wrap
